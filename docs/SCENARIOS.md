# Scenario Content Rules

> **What this file is:** Because Fold Faster is a *content* product, this doc keeps drills consistent: one clear lesson per scenario, beginner-appropriate language, no solver-speak.

## Format

Each scenario includes:

- hole cards
- position
- prior action
- correct action
- short explanation
- leak tag if wrong

## Modules

1. Starting Hands Basics
2. Position Basics
3. Facing a Raise
4. Beginner Traps

## Content rules

- Focus on beginner decisions only
- No advanced balancing
- No solver language
- Explanations should be 1–3 short sentences
- Each scenario should teach one clear lesson

## Explanation style

- Plain English; define any poker term the first time it matters
- Tie the “why” to position, hand strength, or facing aggression—avoid generic slogans

## Difficulty progression

- Stay at **beginner** difficulty until milestones explicitly add intermediate content
- Within a module, order scenarios from clearer decisions to slightly trickier (still beginner-safe)
