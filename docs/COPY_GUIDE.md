# Copy Guide

> **What this file is:** Defines how Fold Faster *sounds*—plain, encouraging, never snobby—so UI text, empty states, and feedback wrappers stay on-brand.

## Tone

- Plain English
- Friendly
- Encouraging
- Never poker-snobby
- Never overly technical

## Good examples

- "This hand looks tempting, but it's too weak from this position."
- "Folding here saves chips in the long run."
- "Good raise — strong hands want to build the pot."

## Avoid

- Dense poker jargon
- Solver-style language
- Condescending coaching tone
- Long explanations
