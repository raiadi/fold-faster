# Vite + React + Tailwind Starter

This is a simple starter template using Vite, React, and Tailwind CSS.

## 🚀 Quick Start

```bash
# 1. Unzip this project
# 2. Install dependencies
npm install

# 3. Run the dev server
npm run dev
```

## ✅ Ready to clone for new projects

- Keep this repo as your base starter
- Clone it for new apps:
  ```bash
  git clone https://github.com/yourusername/react-vite-tailwind-starter my-new-app
  cd my-new-app
  rm -rf .git
  git init
  npm install
  npm run dev
  ```

Happy coding! 🚀✨
